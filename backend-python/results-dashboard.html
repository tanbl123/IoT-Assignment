How it was evaluated

Two safeguards keep these numbers honest:
  • Leakage-free split — overlapping windows from the same recording never appear in both training and test, so the score isn't inflated by near-duplicates.
  • Subject-independent (LOSO) — the model is trained on 16 people and tested on the 17th, rotated through everyone. This measures generalisation to new users, the realistic case.
A 5-subject run gave 90.9% / 76.3% — nearly identical to the 17-subject result, confirming it's robust, not luck.
